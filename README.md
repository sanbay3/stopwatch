# ストップウォッチ・タイマー

ブラウザだけで動くストップウォッチとカウントダウンタイマーです。HTML・CSS・JavaScript のみで構成されています。

## 機能

- **ストップウォッチ**: スタート／ストップ／リセット、ラップタイム（最大 10 件）
- **タイマー**: 分・秒を指定してカウントダウン。終了時は画面の色とアニメーションで通知
- **タブ切り替え**: もう一方の機能が動いている場合は自動で停止

## 使い方

1. リポジトリをクローンまたはダウンロードする
2. `index.html` をブラウザで開く

ローカルサーバー経由で開いても問題ありません（例: `npx serve .` や VS Code の Live Server など）。

## ファイル構成

| ファイル    | 説明           |
| ----------- | -------------- |
| `index.html` | マークアップ   |
| `style.css`  | スタイル       |
| `script.js`  | 動作ロジック   |
